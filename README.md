#Numerical Methods for Quadratic Programming 


All methods are evaluated on carefully designed quadratic programming benchmark problems with known analytical solutions whenever possible. Equality and inequality constrained formulations are considered to validate convergence, feasibility and optimality.

#Requirements:

-Python 3

-NumPy

-SciPy

-Matplotlib 

#Running the Notebook:

Clone the repository and open the notebook using Jupyter Notebook or Google Colab. 
Run all cells sequentially to reproduce the numerical experiments and figures. 

#Academic Context:

This work developed as part of my Integrated Master's Thesis in Applied Mathematics and Physical Sciences at the National Technical University of Athens.
The objective of the thesis was the implementation, analysis and comparison of modern numerical methods for constrained quadratic programming problems.
